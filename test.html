Andradite A Jetpack for New Programmers

It is source-to-source, making it fast as a runtime. Andradite includes a compiler (which can be run in any browser), a corelib and runtime implementation. The corelib/runtime is also very small.

Getting Started

Overview

Run this code in your browser 

class User
  attr_accessor :name

  def initialize(name)
    @name = name
  end

  def admin?
    @name == 'Admin'
  end
end

user = User.new('Bob')
puts user
puts user.admin?

Installation

Install Andradite from RubyGems:

$ gem install Andradite

Or include it in your Gemfile for Bundler:

gem 'Andradite'
